=====================================================
 cadnano: simple oligo analysis and primer design
=====================================================

**cadnano** is DNA nanostructure design software

Importable::

    >>> import cadnano

works in ipython::

    In [1]: import cadnano

dependencies:

    https://pypi.python.org/pypi/PyQt5/5.6


For documentation of the bindings, see https://cadano.github.io/cadnano
